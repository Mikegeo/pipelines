Summary of genomic pipelines

This git repository contains RNA-seq pipelines for transcriptome assembly, gene expression, gene annotation, and other. All scripts are running on XSEDE supercomputers and LIred cluster in the US, including Greenfield, Bridges, and LIred servers.
